# IF1220-Implementasi-L1-Caching-AI-Agents

Repositori ini berisi kode sumber Python untuk mendemonstrasikan analisis kompleksitas algoritma pencarian linear dan hash map dalam kasus caching pada arsitektur AI agents. 
Repositori ini merupakan lampiran dari makalah IF1220 Matematika Diskrit Semester II Tahun Akademik 2025/2026.

## Cara Menjalankan Program 

Pastikan python sudah terinstal di sistem anda. 
1. Buka terminal.
2. Masuk ke direktori `src/`.
3. Jalankan perintah berikut:
